# BSRbearia Thiago Carvalho

Site moderno da barbearia com menu hambúrguer, design escuro premium e integração Ageenda.

## Deploy
1. Fork este repositório
2. Ative GitHub Pages em Settings > Pages > Deploy from branch: main
3. Site ao vivo em: https://seuusuario.github.io/bsrbearia-thiago

## Personalizar
- Substitua imagens em `/img/`
- Atualize WhatsApp no `index.html`
- Edite serviços e preços nas cards
